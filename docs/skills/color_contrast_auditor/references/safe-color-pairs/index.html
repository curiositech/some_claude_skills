Problematic Combinations to Avoid‚Äã

Red + Green (Color Blindness Risk)‚Äã

‚ùå AVOID:
   Background: #4CAF50 (green)
   Text: #F44336 (red)
   Issue: Indistinguishable for deuteranopia

‚úÖ INSTEAD:
   Use icons alongside color
   Or use blue/orange instead of red/green

Yellow on White‚Äã

‚ùå NEVER:
   Background: #FFFFFF
   Text: #FFEB3B (yellow)
   Ratio: 1.1:1 (essentially invisible)

‚úÖ INSTEAD:
   Use dark yellow/gold: #F57F17 ‚Üí Ratio: 3.5:1 (large text only)
   Or use amber: #FF8F00 ‚Üí Ratio: 3.4:1 (large text only)
   For body text, use brown: #5D4037 ‚Üí Ratio: 8.0:1

Light Blue on White‚Äã

‚ùå AVOID:
   Background: #FFFFFF
   Text: #03A9F4 (light blue)
   Ratio: 2.5:1

‚úÖ INSTEAD:
   Use: #0277BD ‚Üí Ratio: 5.4:1
   Or: #01579B ‚Üí Ratio: 7.9:1

Purple on Blue‚Äã

‚ùå AVOID:
   Background: #3F51B5 (indigo)
   Text: #9C27B0 (purple)
   Ratio: 1.5:1

‚úÖ INSTEAD:
   Text: #FFFFFF ‚Üí Ratio: 5.9:1
   Text: #E1BEE7 ‚Üí Ratio: 4.8:1

Brand Color Accessibility Fixes‚Äã

When Client Insists on Brand Colors‚Äã

Scenario: Brand color is #FF6B6B (coral) and they want it on white.

Original: #FF6B6B on #FFFFFF = 2.8:1 ‚ùå

Options:
1. Darken the coral: #D32F2F ‚Üí 5.6:1 ‚úì
2. Keep coral for large text only (‚â•24px): 2.8:1 passes 3:1? No, still fails
3. Use coral for decoration only, use #C62828 for text
4. Add dark background behind coral text

Scenario: Brand uses light gray (#BDBDBD) for "elegant" text.

Original: #BDBDBD on #FFFFFF = 1.9:1 ‚ùå

Options:
1. Darken to #757575 ‚Üí 4.6:1 ‚úì (still "gray", but readable)
2. Darken to #616161 ‚Üí 5.7:1 ‚úì (better)
3. Use #BDBDBD only for decorative elements, not text

CSS Custom Properties Template‚Äã

:root {
  /* Accessible text colors on white */
  --text-primary: #1A1A1A;      /* 16.1:1 */
  --text-secondary: #595959;    /* 7.0:1 */
  --text-tertiary: #767676;     /* 4.5:1 - minimum */
  --text-link: #0066CC;         /* 5.3:1 */
  --text-error: #C62828;        /* 6.0:1 */
  --text-success: #2E7D32;      /* 5.1:1 */
  --text-warning: #E65100;      /* 4.5:1 */
