Make Claude an Expert at Anything

148 free, open-source skills for Claude Code

Claude Code is an AI assistant made by Anthropic. It can read your files, run commands, and write code directly on your computer. Skills are instruction files that give Claude deep expertise in specific areas. Think of it as having a senior developer on retainer who can now selectively go get a master's degree in some new topic.

This collection includes skills for ML engineers (computer vision, drone systems), designers (typography, color theory, UI), founders (career storytelling, competitive analysis), and personal growth (ADHD coaching, Jungian psychology). Each one was built for real projects I've worked on.

Why This Exists

The problem: Every time you ask Claude for help with something specialized, you have to explain the context from scratch. "I'm building a photo app, and I need help with color theory..." Then you get generic advice that misses the nuances.

The solution: Skills pre-load Claude with deep expertise. When you activate the Color Theory Expert skill, Claude already knows about LAB color spaces, earth-mover distance for palette matching, warm/cool alternation‚Äîstuff that would take pages to explain.

Built by Erich Owens. These skills are what I actually use‚Äînot toy examples, but production-grade prompts for speech pathology AI, computer vision, psychology chatbots, and developer tooling.

Who is this for? Developers who use Claude Code. Founders building AI features. Anyone who's ever thought "I wish Claude just knew this stuff already." MIT licensed. Fork it, extend it, make it yours.
